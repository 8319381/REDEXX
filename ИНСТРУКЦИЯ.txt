БИРЖА ЛОГИСТИЧЕСКИХ СТАВОК
======================

Запуск приложения
----------------
1. Установка зависимостей:
   - В папке backend: npm install
   - В папке frontend: npm install

2. Запуск серверов:
   - Запуск backend: 
     cd backend
     npm start
     (сервер запустится на порту 3001)

   - Запуск frontend:
     cd frontend
     npm start
     (приложение откроется на http://localhost:3000)

Работа с приложением
-------------------
1. Регистрация:
   - Выберите роль: логист или покупатель
   - Укажите email и пароль

2. Возможности логиста:
   - Создание ставок по маршрутам
   - Просмотр встречных предложений от покупателей
   - Отслеживание изменений стоимости и сроков

3. Возможности покупателя:
   - Просмотр всех ставок логистов
   - Создание встречных предложений
   - Отслеживание своих предложений по каждой ставке

Маршруты и транспорт
------------------
- Доступные маршруты: Москва – Пекин, Москва – Шанхай
- Тип транспорта: Ж/Д

Системные требования
------------------
- Node.js 14+ и npm
- Современный веб-браузер
- Порты 3000 и 3001 должны быть свободны
